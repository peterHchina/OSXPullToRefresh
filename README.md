# OSXPullToRefresh
for Mac OS X NSScrollView Pull to Refresh

usage:

You can simply indicate NSScrollView Class with "OSXPullToRefreshScrollView" in XIB.

Or use cocoapods  'pod OSXPullToRefresh'
